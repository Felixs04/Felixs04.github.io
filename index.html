Elementos HTML y CSS más usados

HTML (HyperText Markup Language) y CSS (Cascading Style Sheets) son la base de la creación de páginas web. Algunos de los elementos más usados son:

    Elementos estructurales: <div>, <span>
    Encabezados: <h1> a <h6>
    Elementos de texto: <p>, <a>, <strong>, <em>
    Listas: <ul>, <ol>, <li>
    Tablas: <table>, <tr>, <td>
    Formularios: <form>, <input>, <button>
    Multimedia: <img>, <video>, <audio>
